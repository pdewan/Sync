This is a system based on Jon Munson's thesis. It is a client-server based sysem in which clients can synchronize distributed versions of Java objects through the Sync server.

See the following sites for documentation.

The early version of it is described in:

 http://www.cs.unc.edu/~dewan/sync/

The latest version is described through recorded PPT slides:

http://www.cs.unc.edu/~dewan/734/current/lectures/5_Replicated.pdf
